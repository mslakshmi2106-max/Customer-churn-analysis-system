Project Overview

This project focuses on analyzing customer data to understand churn behavior using Python. Synthetic data is generated and analyzed to identify patterns related to customer retention.

Objectives

Generate synthetic customer dataset
Analyze customer attributes like age, balance, and credit score
Identify churn patterns
Perform basic data analysis
Visualize insights using graphs

Tools Used

Python, Pandas, NumPy, Matplotlib

Dataset Description

The dataset includes Customer ID, Age, Gender, Tenure, Balance, Credit Score, Estimated Salary, Number of Products, Active Member status, and Churn status.

Methodology

1. Generate dataset using NumPy
2. Load data into Pandas DataFrame
3. Perform basic analysis
4. Visualize patterns and trends
   
Conclusion

This project helps in understanding customer churn behavior and provides insights for better decision-making.
